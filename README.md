# controls

Little CLI to launch series of controls:

* SSH
* Get zabbix Problems
* Get connections number of postgresql Database
* Check HTTP requests

The CLI prints controls on stdout and save it in a PDF report.

## Example

Configuration file :

```bash
# get an example of configuration
controls -config
```
